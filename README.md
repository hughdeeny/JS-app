# Bula Boys Surf Tours Fiji

Mobile-first Astro website for Bula Boys Surf Tours Fiji — local surf trips to Cloudbreak from Nabila Village, with snorkelling, turtle tours, fishing, spearfishing, and island hopping.

## Development

```sh
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321).

Background dev server:

```sh
astro dev --background
astro dev stop
astro dev status
astro dev logs
```

## Build

```sh
npm run build
npm run preview
```
